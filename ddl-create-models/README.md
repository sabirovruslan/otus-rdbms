# ddl-create-models

Перед запуском убедится что установлены пакеты docker и docker-compose
В файле .env указаны данные пользователя и БД

## Запуск
```docker-compose up```

## Подключение к клиенту mysql
Из директории выполнить команду
```make client```

## Запуск bash контейнера
Из директории выполнить команду
```make bash```
